## ЗАДАНИЕ 1. ТЕОРИЯ МНОЖЕСТВ

### Задача 1.1
**Условие:**  
Операция над множеством M:  
M ⊕ M = ?  (где ⊕ — симметрическая разность)

**Решение:**  
Симметрическая разность: M ⊕ M = (M \ M) ∪ (M \ M) = ∅ ∪ ∅ = **∅**

**Ответ:** M ⊕ M = **∅** (пустое множество)

---

### Задача 1.2
**Условие:**  
Выражение с множествами A, B, C:  
(B ∩ A) ∪ (B \ C) ∪ (A ∩ C) = ?

**Решение:**

**Алгебраическое упрощение:**
- (B ∩ A) ∪ (B \ C) = (B ∩ A) ∪ (B ∩ C̄)
- = B ∩ (A ∪ C̄)
- Полное выражение: B ∩ (A ∪ C̄) ∪ (A ∩ C)
- = B ∩ A ∪ B ∩ C̄ ∪ A ∩ C

**На диаграмме Эйлера (логическое описание):**  
Заштриховываются области:
1. Пересечение A и B (независимо от C)
2. Часть множества B, лежащая вне C (B ∩ C̄)
3. Пересечение A и C

**Конституэнты (объединение атомов):**  
(A ∩ B ∩ C) ∪ (A ∩ B ∩ C̄) ∪ (A ∩ B̄ ∩ C) ∪ (Ā ∩ B ∩ C̄)

**Ответ:**  
Алгебраическая форма: **B ∩ A ∪ B ∩ C̄ ∪ A ∩ C**  
Нормальная форма: **(A ∩ B) ∪ (B ∩ C̄) ∪ (A ∩ C)**

- Регион 2 (A ∩ B ∩ C̄): 0
- Регион 3 (A ∩ B̄ ∩ C): 0
- Регион 4 (A ∩ B̄ ∩ C̄): 1
- Регион 5 (Ā ∩ B ∩ C): 0
- Регион 6 (Ā ∩ B ∩ C̄): 0
- Регион 7 (Ā ∩ B̄ ∩ C): 0

### Множество №132 в виде конституэнт

**M₁₃₂ = (A ∩ B̄ ∩ C̄)** — только эта область заштрихована

### Множество №211

211₁₀ = 128 + 64 + 16 + 2 + 1 = 2⁷ + 2⁶ + 2⁴ + 2¹ + 2⁰ = **11010011₂**

**M₂₁₁ = (A ∩ B ∩ C) ∪ (A ∩ B̄ ∩ C) ∪ (Ā ∩ B ∩ C̄) ∪ (Ā ∩ B̄ ∩ C̄)**

### Операции

**1) Объединение M₁₃₂ ∪ M₂₁₁:**

M₁₃₂ ∪ M₂₁₁ = (A ∩ B̄ ∩ C̄) ∪ (A ∩ B ∩ C) ∪ (A ∩ B̄ ∩ C) ∪ (Ā ∩ B ∩ C̄) ∪ (Ā ∩ B̄ ∩ C̄)

**2) Пересечение M₁₃₂ ∩ M₂₁₁:**

M₁₃₂ ∩ M₂₁₁ = ∅ (т.к. в M₁₃₂ только (A ∩ B̄ ∩ C̄), а это не входит в M₂₁₁)

**3) Симметрическая разность M₁₃₂ ⊕ M₂₁₁:**

M₁₃₂ ⊕ M₂₁₁ = (M₁₃₂ \ M₂₁₁) ∪ (M₂₁₁ \ M₁₃₂)  
= (A ∩ B̄ ∩ C̄) ∪ [(A ∩ B ∩ C) ∪ (A ∩ B̄ ∩ C) ∪ (Ā ∩ B ∩ C̄) ∪ (Ā ∩ B̄ ∩ C̄)]

## ЗАДАНИЕ 8. ЦИКЛИЧЕСКОЕ КОДИРОВАНИЕ

**Информационная посылка:** 2 + 100 = **102** = **1100110₂**

**Порождающий полином:** G(X) = X³ + X + 1 = **1011**

**Кодовое слово:** m(X)·X³ + r(X) = **1100110100** (10 бит)

**Декодирование:**
- **Без ошибки:** синдром s = 0000 ✓
- **С ошибкой:** синдром указывает позицию
- **Кратная ошибка:** не обнаруживается

**Форма:**
- Посылка 1: Все M суть P (модус A)
- Посылка 2: Некоторые S суть M (модус I)
- Вывод: Некоторые S суть P (модус I)

**Валидность:** ✅ **ВАЛИДНЫЙ силлогизм**

**Проверка диаграмма Эйлера:** M ⊂ P, S ∩ M ≠ ∅ ⟹ S ∩ P ≠ ∅ ✓

## ЗАДАНИЕ 10. ЛОГИКА ВЫСКАЗЫВАНИЙ

**Формализация:** "Если (p ∨ ¬q) то r"  
**(p ∨ ¬q) → r**

### СДНФ: (¬p ∧ q) ∨ (p ∧ r)

### СКНФ: (p ∨ q ∨ r) ∧ (p ∨ ¬q ∨ r) ∧ (¬p ∨ ¬q ∨ r)

### Дизъюнктивное разложение Шеннона:
**(p ∧ r) ∨ (¬p ∧ q) ∨ (¬p ∧ r)**

### Конъюнктивное разложение Шеннона:
**(p ∨ q ∨ r) ∧ (¬p ∨ r)**
