#### ASCII‑диаграмма Эйлера для A, B, C

Условно изобразим три множества:

- A — горизонтальный овал слева
- B — горизонтальный овал справа
- C — круг внутри/пересекающий A и B

|               |
;   A∩B   C̄    ;

Где:

Заштриховать A∩B (вся область пересечения A и B, в том числе с C).

Заштриховать часть B вне C (B ∩ C̄).

Заштриховать A∩C (пересечение A и C, включая A∩B∩C).
